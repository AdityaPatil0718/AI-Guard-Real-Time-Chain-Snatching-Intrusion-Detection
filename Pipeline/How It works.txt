üìÑ Project: Activity Detection and Frame Analysis Pipeline
This pipeline processes an input video to detect activities (e.g., snatching, climbing), identifies the frame with the highest confidence for the detected activity, and then performs face detection and description analysis using Google's Gemini API.

‚úÖ Pipeline Overview (main_pipeline.py)
The following steps summarize what the script does:

Load Input Video:

Specify the path to the input video in the script.

Activity Detection:

The script uses the YOLOv8 model (via the ultralytics library) to detect activities in the video.

It identifies and saves:

The frame with the highest confidence.

The previous frame.

The next frame.

Face Detection and Cropping:

Uses the FaceTorsoExtractor class from the main_face.py module.

This extracts and crops the face (or torso) region from the best frame.

Description Analysis via Gemini API:

Uses the GeminiAPI class from the gemini_api.py module.

The best frame is uploaded to Gemini for detailed analysis.

Requires a valid Gemini API key to be set.

üìÇ File Descriptions
main_pipeline.py: The main driver script for the entire pipeline.

main_face.py: Contains the FaceTorsoExtractor class for face/torso detection and cropping.

gemini_api.py: Contains the GeminiAPI class used to send frames to Gemini and get descriptive analysis.

üìù Output
